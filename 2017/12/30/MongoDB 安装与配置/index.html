MongoDB安装与配置

Author: jianjian Date: December 30, 2017  15:12:30
MongoDB版本：3.4 # 一、下载MongoDB

官网下载地址：https://www.mongodb.com/download-center#enterprise

二、配置MongoDB

有安装版和解压版提供选择，安装后目录结构相同，如下：

  • /mongodb
    • /bin
    • GNU-AGPL-3.0
    • MPL-2
    • README
    • THIRD-PARTY-NOTICES
将bin目录配置到系统变量path中 编写mongoDB的配置文件（mongodb.config） 文件内容： 将mongodb服务添加到系统中 以管理员启动cmd，输入：

mongod –config D:\devSoft\mongodb\mongodb.config –install –serviceName “mongodb”

这里将–config后的参数配置为上一步所写配置文件的地址

这里将mongodb服务命名为‘mongodb’

使用命令启动和关闭mongodb服务 net start mongodb //启动服务器

net stop mongodb //关闭服务器

三、MongoDB可视化工具推荐

  • MongoDB Cloud Manager
  • MongoClient
  • adminMongo
  • NoSQL Manager for MongoDB
  • Robo 3T
